Switch User sample

Demonstrates an approach to allow switching between multiple users on, for example, a shared family iPad.

Build Requirements
iOS 4.0 SDK

Runtime Requirements
iPhone OS 4.0 or later

Using the Sample
Install the Facebook iOS SDK.
Launch the SwitchUsersSample project using Xcode from the <iOS SDK>/samples/SwitchUsersSample directory.
